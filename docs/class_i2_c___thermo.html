Detailed Description

Base class for thermal sensors which use an I2C bus to communicate with a microcontroller.

The I2C thermal sensors need to have an I2C bus object (such as Wire) to which to talk, and they each have an address on that bus, so these are saved as class member data. Methods are provided to read and write data to the sensor using I2C. A possibly helpful method is provided which can scan the I2C bus and show the addresses at which devices have been found.

Set up the base class for I2C temperature sensors.

This constructor does things which all I2C temperature sensors need to do. Child classes will do device specific setup. The parent class's constructor is called on the :BaseThermometer() line; the parent class constructor doesn't take any parameters.

Parameters
p_i2cA pointer to an I2C ("TwoWire" in Arduino-ese) object which has already been set up (created and its .begin() method run)
i2c_addressThe address of a specific sensor on the I2C bus
