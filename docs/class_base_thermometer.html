virtual float BaseThermometer::celsius ( void  )

Pure virtual method to get temperature in degrees Celsius.

This method is declared and set "equal" to zero, meaning it doesn't exist and must be declared in child classes. As this method doesn't exist, one cannot create an object of class BaseThermometer; one must create objects of child classes which override this method with something that works.

The measured temperature in degrees Celsius

◆ fahrenheit()

virtual float BaseThermometer::fahrenheit ( void  )
inlinevirtual

Return the measured temperature in degrees Fahrenheit.

Most sensors return Celsius temperature by default, so this method gets the Celsius temperature and converts it. Child classes can just use this method and not creeate their own. If a sensor measures Fahrenheit directly, it can override this method. This method is declared inline, meaning instead of it being called as a function (with saving stuff on the stack and branching and returning), its code will replace the function call wherever this function is called. That makes things run a little bit faster but may take a bit of extra program space.

Returns
The measured temperature in Fahrenheit
